## JAVA와 C/C++의 차이점

#### JAVA 특징

- 객체지향 언어
- 보안, 이식성, 빠른 개발에 비중을 둠
- 인터프리터와 컴파일러를 모두 사용

#### C++

- 객체지향 언어
- 타입체크에 엄격
- 실행시간의 효율성 저하를 최소화 -> 인라인 함수의 도입 등 함수 호출로 인한 성능 저하를 막음
- 게임 개발에 많이 사용

#### C

- 절차지향 언어
- 컴파일러 언어 -> 인터프리터 언어에 비해 실행 속도가 빠름
- Low Level 언어의 특징을 가짐
- 시스템 자원을 개발자가 직접 제어할 수 있음
